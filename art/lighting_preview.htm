Lighting Preview

 

 

The in-game lighting preview tool can be invoked by bringing up the console and typing “pushmenu lighting”. The lighting preview tool shows up at the bottom of the screen and looks like the diagram below. Sliders can be moved and the effect of changing the values is displayed in the game.

 

 

 

 

Here is a description of each of the controls:

 

Dir

The direction of the real-time light (the sun). The first number is an angle of elevation from the horizon, and the second number is a compass degree, where 0 = north.

 

Diffuse

The color of the real-time light. Diffuse light increases with the angle of the light hitting a surface.

 

Ambient

The color of the ambient light. Ambient light is a color that is added equally to all objects.

 

Close

Click to close the lighting tool.

 

SaveFX

Save the results to a file.

 

GAME

Uses the game settings.

 

TWEAK

Allows you to modify the current settings.

 

OFF

Turns off all postfx (full screen effects).

 

Farplane

The distance of the farplane from the camera.

 

Bias

Controls the blending of the farplane with objects in the scene.

 

CC_Sat

The postfx saturation multiplyer. 1 = fully saturated.

 

CC_Blur

The amount of postfx blur

 

CC_Add

The postfx color which is added to the entire screen

 

CC_Mul

The postfx color which is multiplied with the entire screen.

 

Color

The color of the farplane.
