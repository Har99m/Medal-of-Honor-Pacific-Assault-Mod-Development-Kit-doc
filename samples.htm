Samples

 

            The following samples are installed in your “My Documents\EA Games\Medal of Honor Pacific Assault(tm)\moddir” folder:

 

MapSample

This sample includes a simple multiplayer map to show how to create a multiplayer game. It goes with the Level Design tutorial.

 

WeaponSample

The weapon sample includes a “shooting range” map which can be used for testing weapon mods. Refer to the associated Weapon Tutorial to see how this mod is used.

 

MultiplayerSample

An actual multiplayer map from Medal of Honor Pacific Assault! This is the “Bataan” level, and can be used as an example of how to create a full featured multiplayer level.
