MOHRadiant

          This section describes the map editor for Medal of Honor Pacific Assault, MOHRadiant. MOHRadiant is based on the Quake editor Radiant. MOHRadiant is designed to use 3D models created in a separate modeling program. You can also create detailed terrain in MOHRadiant. Both of these features make it better suited for creating the outdoor environments used in Medal of Honor Pacific Assault.

 

MilkShape3D

          MilkShape3D is the editor supported by the MDK. An unregistered version is included with the MDK installation. Also included are the necessary MilkShape plugins for creating 3D models compatible with Medal of Honor Pacific Assault.
